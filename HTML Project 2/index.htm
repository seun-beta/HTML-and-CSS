An Overview of CSS

These pages are intended to be explored in the developer console of your browser. You should also install the Web Developer plugin from Chris Pedrick to play with CSS.

We can use style= attribute on HTML tags to apply various styles to text and blocks of text.

The "Cascading" in CSS means that the closest style setting (monospace) "wins" over a style setting (arial) that is "further away".

We can put a border around a block of text. You probably want a CSS sheet handy or use Google search to find the CSS values.
