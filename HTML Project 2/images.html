CSS and Images

It is fun to float an image to the left or right and watch the text wrap around it. You might want to add a bit of margin so the text does not crowd it. An em is the "height" of the current "font" so it sclaes when the size of the font is changed.


Sometimes we need to clear the "hanging wrap" when we want the next paragraph to go all the way to the margin - in effect "clearing the float"

Images can be right in the middle of the text like a character. And we can even make an image a clickable link to the starting page.
